* 起源
```
local _gcd
_gcd = function (a,b)
    if b == 0 then
        return a
    end
    
    return _gcd (b, a % b)
end
```

* Lua具有这么几个特点
```
    这是一种动态类型语言，常用习惯里变量不需要定义类型，直接给变量赋值就行。
    Lua在对一个数字字符串进行算术操作时，会尝试将这个数字字符串转成一个数字。
    在Lua语言中，数学运算n%0的结果是nan(Not A Number)。
```
