* 状态码
```
| 正常状态码 | 描述       |
| ---------- | ---------- |
| 200        | OK         |
| 201        | Created    |
| 202        | Accepted   |
| 204        | No Content |

| 错误状态码 | 描述                     |
| ---------- | ------------------------ |
| 400        | Bad Request              |
| 401        | Unauthorized             |
| 403        | Forbidden                |
| 404        | Not Found                |
| 405        | Method Not Allowed       |
| 409        | Conflict                 |
| 413        | Request Entity Too Large |
| 415        | Unsupported Media Type   |
| 500        | Internal Server Error    |
| 501        | Not Implemented          |
| 503        | Service Unavailable      |
```
